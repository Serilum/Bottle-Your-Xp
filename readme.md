<p><strong><span style="font-size:24px">Requires the library mod&nbsp;<a style="font-size:24px" href="https://curseforge.com/minecraft/mc-mods/collective" rel="nofollow">Collective</a>.</span></strong><strong>&nbsp;<br><br> &nbsp; &nbsp;This mod is part of <span style="color:#008000"><a style="color:#008000" href="https://curseforge.com/minecraft/modpacks/the-vanilla-experience" rel="nofollow">The Vanilla Experience</a></span>.</strong><br><span style="font-size:18px">Bottle Your Xp is a minimalistic configurable mod which allows the creation of experience bottles (bottles o' enchanting) from your own gathered xp by <strong>sneak/crouch + right-clicking</strong>&nbsp;an empty bottle. This can optionally (set in the config) hurt the player because the procedure might not be entirely painless.</span><br><br><br><strong><span style="font-size:20px">Configurable:</span> <span style="color:#008000;font-size:14px"><a style="color:#008000" href="https://github.com/Serilum/.information/wiki/how-to-configure-mods" rel="nofollow">(&nbsp;how do I configure?&nbsp;)</a></span><br></strong><strong>damageOnCreation</strong>&nbsp;(default = true): If enabled, damages the user whenever they create an experience bottle.<strong><br>halfHeartDamageAmount </strong>(default = 1, min 1, max 20):The amount of damage the user takes times half a heart when creating an experience bottle.<strong><br>rawXpConsumedOnCreation </strong>(default = 10, min 0, max 100): The amount of raw xp it takes to produce an experience bottle. Be careful when setting this too low, as it may enable xp duplication.<br><br><br><strong>The effect in action:<br></strong></p>
